# 交互设计方案

## 5.1 全局交互

| 场景         | 交互行为                | 反馈                                       |
| :----------- | :---------------------- | :----------------------------------------- |
| **页面加载** | 骨架屏渐进加载          | 卡片按顺序淡入，延迟 100ms 级联            |
| **数据刷新** | 点击刷新按钮或定时触发  | 按钮旋转动画，数据平滑过渡（数字滚动效果） |
| **图表联动** | 点击某图表的图例/数据项 | 其他关联图表高亮相同维度，支持配置联动规则 |
| **全屏查看** | 点击卡片全屏按钮        | 卡片展开为模态框，图表重绘适配新尺寸       |
| **导出报表** | 顶部导出按钮            | 生成 PDF/PNG，带时间戳水印                 |

## 5.2 配置面板交互

```
配置流程:
1. 点击画布空白处 / 点击已有图表
   ↓
2. 右侧面板切换为对应配置模式
   - 新建模式: 先选图表类型 → 再配数据源 → 映射字段
   - 编辑模式: 显示当前图表所有可配置项
   ↓
3. 实时预览: 配置修改后立即在画布更新（防抖 300ms）
   ↓
4. 确认/取消: 支持撤销重做（Ctrl+Z / Ctrl+Shift+Z）
```

