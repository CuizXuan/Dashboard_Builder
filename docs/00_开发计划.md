# Dashboard_Builder 开发计划

## 项目概述

**智能数据可视化看板平台** — 面向业务人员的低代码数据展示工具。

---

## 开发阶段总览

| 阶段 | 名称 | 周期 | 交付物 |
| :--- | :--- | :--- | :--- |
| 阶段一 | 核心框架 | 2周 | 布局系统 + 状态管理 + 数据源面板 + 图表卡片容器 |
| 阶段二 | 图表引擎 | 3周 | ECharts 集成：饼图、折线图、柱状图、表格 + 图例联动 |
| 阶段三 | 配置系统 | 2周 | 右侧属性面板 + 数据映射可视化配置 + 实时预览 + 撤销重做 |
| 阶段四 | 高级功能 | 2周 | 图表联动与钻取 + 响应式适配 + 主题系统 |
| 阶段五 | 优化交付 | 1周 | 性能优化 + 无障碍访问 + 导出功能 |

---

## 阶段一：核心框架（2周）

### 目标
搭建项目骨架，实现三栏布局、状态管理、基础数据源管理。

### 目录结构
```
src/
├── main.tsx                    # 应用入口
├── App.tsx                     # 根组件
├── components/
│   ├── Layout/
│   │   ├── Header.tsx          # 顶部导航栏
│   │   ├── LeftPanel.tsx       # 左侧数据源面板
│   │   ├── Canvas.tsx          # 看板画布区域
│   │   ├── RightPanel.tsx      # 右侧配置面板
│   │   └── StatusBar.tsx       # 底部状态栏
│   └── common/
├── store/
│   └── useDashboardStore.ts   # Zustand 状态管理
├── styles/
│   └── global.css              # 全局样式 + CSS变量
└── types/
    └── index.ts                # TypeScript 类型定义
```

### 技术决策
- React 18 + TypeScript
- Vite 构建工具
- Zustand 状态管理
- React-Grid-Layout 画布布局
- Ant Design 5 基础组件

### 验收标准
- [ ] 三栏布局渲染正常，响应式断点切换正确
- [ ] 可拖拽调整卡片位置和大小
- [ ] 数据源面板可添加/编辑/删除 API 和静态 JSON 数据源
- [ ] 状态栏显示图表数量和最后更新时间
- [ ] 布局配置可持久化到 localStorage

---

## 阶段二：图表引擎（3周）

### 目标
集成 ECharts 5，实现四种基础图表类型的完整渲染和交互。

### 新增目录
```
src/
├── components/Charts/
│   ├── ChartCard.tsx           # 图表卡片容器
│   ├── charts/
│   │   ├── PieChart.tsx        # 饼图/环形图
│   │   ├── LineChart.tsx       # 折线图/面积图
│   │   ├── BarChart.tsx        # 柱状图/条形图
│   │   └── TableChart.tsx      # 数据表格
│   └── LegendPanel.tsx         # 图例组件
├── hooks/
│   └── useChartData.ts         # 图表数据转换 Hook
└── utils/
    └── chartConfig.ts          # ECharts 主题配置
```

### 验收标准
- [ ] 四种图表类型正常渲染，数据正确映射
- [ ] 图例点击可显示/隐藏对应数据系列
- [ ] 图例 Hover 高亮对应图表元素，其他元素透明度降低
- [ ] 图表支持全屏查看
- [ ] 卡片支持刷新当前图表数据

---

## 阶段三：配置系统（2周）

### 目标
完整实现右侧配置面板，支持图表类型选择、数据映射、样式设置。

### 新增/修改目录
```
src/
├── components/
│   └── RightPanel/
│       ├── ChartTypeSelector.tsx   # 图表类型选择
│       ├── DataMappingPanel.tsx     # 数据映射配置
│       ├── StylePanel.tsx           # 样式设置面板
│       └── InteractionPanel.tsx     # 交互行为配置
├── hooks/
│   └── useDataSource.ts             # TanStack Query 数据请求
└── store/
    └── useDashboardStore.ts        # 新增撤销/重做逻辑
```

### 验收标准
- [ ] 点击画布空白处 → 新建图表流程完整
- [ ] 点击已有图表 → 编辑模式配置项完整
- [ ] 配置修改后 300ms 防抖实时预览
- [ ] 支持 Ctrl+Z 撤销 / Ctrl+Shift+Z 重做
- [ ] 数据源支持轮询间隔配置

---

## 阶段四：高级功能（2周）

### 目标
图表联动、响应式优化、主题系统。

### 验收标准
- [ ] 支持配置图表间联动（点击某图表数据项 → 其他图表高亮）
- [ ] 平板（768-1199px）布局正确
- [ ] 手机（<768px）单栏布局正确
- [ ] 支持浅色商务/深色科技/浅色清新三种主题切换
- [ ] 主题配置持久化
