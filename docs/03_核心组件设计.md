# 核心组件设计

## 3.1 图表卡片组件 (ChartCard)

```
┌────────────────────────────────────────┐
│  [图标] 卡片标题                    [⋮] │  ← 头部 (40px, 标题左对齐, 菜单右对齐)
├────────────────────────────────────────┤
│                                        │
│         ┌──────────────┐               │
│         │              │   条目    值   百分比 │
│         │    饼图      │   ● 未设定  105  100% │
│         │   (自适应)    │   ─────────────────  │
│         │              │                    │
│         └──────────────┘                    │
│                                        │
│  注：图例在右侧，与图表水平排列              │
├────────────────────────────────────────┤
│  更新时间: 16:35:22    [刷新] [全屏]      │  ← 底部工具栏 (32px)
└────────────────────────────────────────┘
```

**卡片样式:**
- 背景: #FFFFFF
- 圆角: 8px
- 阴影: `0 2px 8px rgba(0,0,0,0.06)`
- 内边距: 16px
- 悬停阴影: `0 4px 16px rgba(0,0,0,0.12)`
- 过渡: `box-shadow 0.3s ease`

## 3.2 图例组件 (LegendPanel)

| 元素         | 设计                                                         |
| :----------- | :----------------------------------------------------------- |
| **布局**     | 右侧垂直列表，与图表区域水平排列（图表:图例 ≈ 6:4 或 7:3）   |
| **条目高度** | 36px/行，保证可点击区域                                      |
| **指示器**   | 左侧 8px 圆点，与图表颜色严格对应                            |
| **文字区**   | 条目名称（左对齐）+ 数值（右对齐，等宽字体）+ 百分比（最右，灰色） |
| **交互**     | Hover 整行高亮 + 对应图表扇区放大/高亮；点击可隐藏/显示该数据系列 |
| **滚动**     | 超过 6 条时显示自定义滚动条（6px宽，圆角，灰色轨道）         |

**图例行结构:**
```
┌────────────────────────────────────┐
│ ●  联合调度决策/防洪调度...   27   25.71% │
│    ↑8px圆点  ↑左对齐，溢出省略  ↑右对齐  ↑灰色 │
└────────────────────────────────────┘
```
**Hover态:** 背景 #F5F5F5，左侧出现 2px 主色竖线指示

## 3.3 数据源配置面板

```
┌─────────────────┐
│ 数据源管理       │
│ ─────────────── │
│ ▼ 已连接源 (3)   │
│   ┌───────────┐ │
│   │ ● API接口  │ ← 绿色圆点表示在线
│   │   bug_api │   灰色表示离线，红色表示错误
│   │   [配置]  │ │
│   └───────────┘ │
│   ┌───────────┐ │
│   │ ● 静态JSON│ │
│   │   demo_data│ │
│   └───────────┘ │
│ ─────────────── │
│ [+ 新建数据源]   │
│   ┌───────────┐ │
│   │ API接口    │ │
│   │ 数据库     │ │
│   │ Excel/CSV  │ │
│   │ 静态JSON   │ │
│   └───────────┘ │
└─────────────────┘
```

**数据源类型定义:**

| 类型          | 配置项                                     | 数据刷新     |
| :------------ | :----------------------------------------- | :----------- |
| **REST API**  | URL、请求方法、Headers、认证方式、轮询间隔 | 支持定时轮询 |
| **GraphQL**   | Endpoint、Query语句、变量                  | 支持订阅     |
| **静态JSON**  | 直接粘贴/上传文件                          | 手动刷新     |
| **WebSocket** | URL、连接参数、重连策略                    | 实时推送     |
| **数据库**    | 连接字符串、SQL查询（仅服务端代理）        | 定时轮询     |
