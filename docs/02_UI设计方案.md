# UI 设计方案

## 2.1 整体布局结构

采用 **"三栏经典布局"** 变体：

```
┌──────────────────────────────────────────────────────────────┐
│  [Logo]  数据看板平台    [全局刷新] [导出报表] [用户头像]        │  ← 顶部导航栏 (48px)
├──────────────┬─────────────────────────────┬─────────────────┤
│              │                             │                 │
│  数据源      │                             │   图表配置      │
│  管理面板    │      看板画布区域            │   属性面板      │
│  (240px)    │      (自适应剩余宽度)        │   (280px)      │
│              │                             │                 │
│  ┌────────┐ │  ┌─────────────────────┐    │  ┌───────────┐ │


│  │ 数据源A │ │  │                     │    │  │ 图表类型  │ │
│  │ 数据源B │ │  │    图表卡片网格      │    │  │ 样式设置  │ │
│  │ 数据源C │ │  │                     │    │  │ 数据映射  │ │
│  └────────┘ │  │  ┌─────┐ ┌─────┐    │    │  │ 交互行为  │ │
│              │  │  │饼图 │ │折线图│    │    │  └───────────┘ │
│  [+添加源]   │  │  └─────┘ └─────┘    │    │                │
│              │  │  ┌─────┐ ┌─────┐    │    │  [应用] [重置] │
│              │  │  │柱状图│ │表格 │    │    │                │
│              │  │  └─────┘ └─────┘    │    │                │
│              │  └─────────────────────┘    │                │
│              │                             │                │
├──────────────┴─────────────────────────────┴────────────────┤
│  状态栏: 最后更新: 2026-04-21 16:35 | 图表数量: 3 | 数据正常    │  ← 底部状态栏 (32px)
└──────────────────────────────────────────────────────────────┘
```

## 2.2 色彩体系

| 用途           | 色值                                                         | 说明                                    |
| :------------- | :----------------------------------------------------------- | :-------------------------------------- |
| **主背景**     | `#F0F2F5`                                                    | 浅灰蓝，护眼且突出内容卡片              |
| **卡片背景**   | `#FFFFFF`                                                    | 纯白，带1px边框 `#E8E8E8`               |
| **主色调**     | `#1890FF`                                                    | Ant Design 蓝，用于按钮、激活态         |
| **图表主色板** | `['#FF6B6B', '#FFD93D', '#6BCB77', '#4D96FF', '#9B59B6', '#E67E22', '#1ABC9C', '#34495E']` | 8色循环 |
| **文字主色**   | `#262626`                                                    | 标题、重要文字                          |
| **文字次色**   | `#595959`                                                    | 正文、图例文字                          |
| **文字辅助**   | `#8C8C8C`                                                    | 时间戳、百分比                          |
| **分割线**     | `#F0F0F0`                                                    | 卡片内部分隔                            |
| **Hover态**    | `rgba(24, 144, 255, 0.05)`                                   | 浅蓝背景反馈                            |

## 2.3 字体规范

| 层级     | 字号 | 字重 | 行高 | 用途                         |
| :------- | :--- | :--- | :--- | :--------------------------- |
| 页面标题 | 20px | 600  | 28px | 看板名称                     |
| 卡片标题 | 16px | 500  | 24px | "迭代Bug数量"、"模块Bug数量" |
| 图表标签 | 12px | 400  | 20px | 饼图扇区标签、图例文字       |
| 数据值   | 24px | 600  | 32px | 核心指标数字（如"105"）      |
| 百分比   | 14px | 400  | 22px | "100%"、"25.71%"             |
| 辅助说明 | 12px | 400  | 20px | 状态栏、提示文字             |

**字体栈**: `-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, "Noto Sans SC", sans-serif`

## 2.4 响应式断点

| 断点     | 宽度        | 布局调整                                 |
| :------- | :---------- | :--------------------------------------- |
| **大屏** | ≥1600px     | 三栏布局，画布区域最大 1200px，居中显示  |
| **桌面** | 1200-1599px | 三栏布局，右侧配置面板可折叠为图标栏     |
| **平板** | 768-1199px  | 左侧数据源面板折叠，配置面板变为底部抽屉 |
| **手机** | <768px      | 单栏垂直堆叠，图表全宽，图例移至图表下方 |
