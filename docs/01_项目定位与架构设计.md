# 项目定位与架构设计

## 1.1 项目定位

**智能数据可视化看板平台** —— 面向业务人员的低代码数据展示工具，支持：

- 自由配置数据源（API、静态JSON、数据库直连）
- 拖拽式图表编排
- 实时数据刷新与交互联动
- 响应式布局适配

## 1.2 技术架构分层

```
┌─────────────────────────────────────────┐
│           展示层 (Presentation)          │
│  ┌─────────┐ ┌─────────┐ ┌─────────┐ │
│  │ 看板画布 │ │ 图表组件 │ │ 配置面板 │ │
│  └─────────┘ └─────────┘ └─────────┘ │
├─────────────────────────────────────────┤
│           逻辑层 (Business Logic)        │
│  ┌─────────┐ ┌─────────┐ ┌─────────┐ │
│  │ 数据源管理│ │ 图表引擎 │ │ 状态管理 │ │
│  └─────────┘ └─────────┘ └─────────┘ │
├─────────────────────────────────────────┤
│           数据层 (Data Access)           │
│  ┌─────────┐ ┌─────────┐ ┌─────────┐ │
│  │ API适配器│ │ 本地存储 │ │ 数据转换 │ │
│  └─────────┘ └─────────┘ └─────────┘ │
└─────────────────────────────────────────┘
```

## 1.3 关键设计决策

| 决策点       | 选择              | 理由                                                   |
| :----------- | :---------------- | :----------------------------------------------------- |
| **图表引擎** | ECharts 5         | 功能完备、图例系统成熟、中文文档友好                   |
| **UI组件库** | Ant Design 5     | 企业级设计体系、表单配置能力强、生态丰富               |
| **状态管理** | Zustand           | 轻量、TypeScript友好、适合中等复杂度应用               |
| **布局系统** | React-Grid-Layout | 成熟的拖拽网格布局，支持响应式断点                     |
| **数据请求** | TanStack Query    | 自动缓存、重试、去重、后台刷新                         |
