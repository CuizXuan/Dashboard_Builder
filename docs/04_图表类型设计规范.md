# 图表类型设计规范

## 4.1 饼图/环形图

**数据映射：**
- 维度字段 → 图例条目名称
- 度量字段 → 扇区角度 + 右侧数值列
- 自动计算百分比 → 右侧百分比列

**视觉规范：**
- 饼图内半径: 外半径 = 0 : 1（实心饼图）或 0.6 : 1（环形图）
- 扇区间隔: 2px 白色缝隙
- 标签位置: 外部引导线（数据密集时）或 内部直接标注
- 最小扇区: 小于 2% 时合并为"其他"

## 4.2 折线图/面积图

- 网格线: 水平虚线 `#E8E8E8`，垂直线可选
- 数据点: 4px 圆点，Hover 放大至 8px 并显示 Tooltip
- 线条粗细: 2px，面积图填充透明度 0.1
- 多系列: 使用色板循环，图例在顶部或右侧

## 4.3 柱状图/条形图

- 柱间距: 柱宽度的 30%
- 圆角: 顶部 4px 圆角（横向条形为右侧圆角）
- 背景柱: 可配置目标值对比（浅灰色背景柱）

## 4.4 表格

- 表头: 背景 `#FAFAFA`，底部 2px 主色线
- 行高: 48px，Hover 背景 `#F5F5F5`
- 分页: 底部居中，每页默认 10/20/50 条可选

## 4.5 图表通用交互

| 操作                 | 效果                                       |
| :------------------- | :----------------------------------------- |
| **Hover 扇区/柱/点** | 放大 + Tooltip 显示详细数据                |
| **Hover 图例行**     | 对应图表元素高亮，其他元素透明度降至 0.3   |
| **点击图例圆点**     | 切换该系列显示/隐藏，圆点变灰并打斜线      |
| **拖拽图例排序**     | 长按拖拽调整图例顺序，图表扇区顺序同步更新 |
